## 一、用户动线总览

```
动线 A（朋友分享进入）：
  好友分享链接 → P1 分享落地页 → 点击 CTA → P3 安装命令弹窗 → P4 等待页 → P5 结果页

动线 B（直接访问进入）：
  访问首页 → P2 官网首页 → 点击 CTA → P3 安装命令弹窗 → P4 等待页 → P5 结果页

动线 C（结果页分享裂变）：
  P5 结果页 → 点击"炫耀成绩" → 复制分享文案（链接指向 /s/{token}）→ 好友打开 → P1 → 动线 A

动线 D（¥19.9 基础能力升级）：
  P5 → 点击"¥19.9 补齐基础能力" → P6 Step1(说明+付费) → Step1.5(扫码支付) → Step2(复制命令) → Step3(等待) → Step4(结果) → 回到 P5（数据刷新）

动线 E（¥99/月 高级能力订阅）：
  P5 → 点击"¥99/月 高级能力订阅" → 登录弹窗（未登录时）→ 跳转 /me/{token} → P8 订阅弹窗 → 扫码支付 → 点击"我已支付" → P7 个人主页

动线 F（个人主页）：
  P7 个人主页（需登录）→ 查看成绩+订阅状态 → 炫耀/重测/高级订阅
```
