# P7 · 个人主页 (`/me/{token}`) — 设计 PRD

**版本**：v1.0 · 2026-03-12
**对应前端**：`templates/me.html`（附 `me-preview.html` 可直接浏览器预览）

---

## 一、页面概述

个人主页是付费用户的核心管理页面。展示龙虾 IQ 成绩、会员权益、技能面板，并承载续费和技能激活流程。

**页面入口**：
- 结果页（P5）点击「¥99/月 高级能力订阅」→ 登录 → 跳转此页
- 直接访问 `/me/{token}`

**前置条件**：需登录（手机号+验证码，MVP 阶段固定验证码 888888）

---

| 属性 | 值 |

## 三、页面状态定义

页面有 **两种状态**，由后端 `payment_status` 决定：

| 状态 | 条件 | 表现 |
|------|------|------|
| **未付费** | `payment_status` 非 `submitted` | 进入页面 0.8s 后自动弹出订阅弹窗 |
| **已付费** | `payment_status == submitted` | 直接展示会员面板，无自动弹窗 |

### 4.3 IQ 环形进度

```
        ╭──── 渐变环 ────╮
       │   [龙虾图片]     │
       │      186         │    ← Outfit 52px 900
       │    智力值         │    ← Outfit 16px 400
        ╰─────────────────╯
```

- SVG 环：半径 85px，描边 6px，渐变 `#CC0F14 → #FF9616`
- 环内：龙虾图片（径向渐变遮罩淡出），数字覆盖在上
- **入场动画**：数字从 0 递增到实际 IQ（1.2s ease-out），环形同步展开
- IQ 最大值 270，环的满度 = IQ/270

## 十、动画定义

| 动画 | 触发 | 参数 |
|------|------|------|
| IQ 数字递增 | 页面加载 0.4s 后 | 0→IQ，1.2s ease-out |

## 十一、数据来源

| 字段 | 来源 | 说明 |
|------|------|------|
| `data.name` | 后端 SQLite | 龙虾名字 |
| `data.iq` | 后端计算 | 原始分 → IQ 映射（30-270） |
| `data.title` | 后端计算 | 基于 IQ 的称号 |
| `data.rank` | 后端查询 | 排行榜排名 |
| `data.model` | 提交时记录 | AI 模型名 |
| `payment_status` | 后端 payments 表 | `None` / `pending` / `submitted` |
| `expire_date` | 后端计算 | confirmed_at + 30 天 |
| `token` | URL 路径参数 | 用户唯一标识 |
