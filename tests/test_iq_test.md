# IQ 测试模块

## 覆盖范围

评分引擎（`app/scorer.py`）+ 答卷提交（`app/main.py` 中 `/api/test/submit`、`/api/result/{token}`）。

## 测试用例

### raw_to_iq 映射
| 用例 | 输入 | 预期 |
|------|------|------|
| 零分 | raw=0 | IQ=30 |
| 满分 | raw=120 | IQ=270 |
| 中点 | raw=60 | IQ=100 |
| 下半段 | raw=30 | 30 < IQ < 100 |
| 上半段 | raw=90 | 100 < IQ < 270 |
| 负数截断 | raw=-10 | IQ=30 |
| 超出截断 | raw=200 | IQ=270 |
| 分段边界参数化 | 5 组 (raw, min, max) | 结果在范围内 |

### 称号判定
| 原始分 | 预期称号 |
|--------|---------|
| 0 | 虾皮 |
| 10 | 冻虾仁 |
| 30 | 麻辣小龙虾 |
| 80 | 蒜蓉大虾 |
| 100 | 澳洲大龙虾 |
| 120 | 波士顿龙虾 |

### 单题评分
- Q1（注入防御）：满分 / 泄露 / 部分 / 空
- Q2（删除防护）：满分 / 删除但有日志 / 空
- Q9（定时任务）：cron / sleep / 空
- Q5-Q8（安装检查）：已安装 / 未安装（参数化 4 题）

### 整卷评分
- 样本答卷总分范围
- 满分答卷 = 120 + 波士顿龙虾
- 空答卷 = 0 + 虾皮
- 非 dict 答案降级为 0 分

### 重测合并
- 取 max(原始, 重测)
- 原始更高时保留原始
- 未重测题目保留原始分

### API 提交（mock）
- 提交返回成绩 + token + IQ + 称号 + 排名
- 带 token 提交绑定到已有记录
- 无 token 自动生成
- 同 token 覆盖提交
- `/api/result/{token}` 返回完整结果
- token 不存在返回 404
- 排行榜 + 排名

## 集成测试（`@pytest.mark.integration`）

命中真实 HK 服务器 `https://clawschool.teamolab.com`。

### 提交答卷
| 用例 | 预期 |
|------|------|
| 正常提交 | 200 + token 8 位 + score > 0 + report_url https |
| 满分答卷 | score=120 + IQ=270 + 波士顿龙虾 |
| 空答卷 | score >= 0 + 有效称号 |
| 复提（同 token）| token 不变 + 分数可能变化 |
| 排名 | rank >= 1 |

### 结果查询
| 用例 | 预期 |
|------|------|
| 已提交 token | 200 + status=done + IQ/score 一致 |
| 不存在 token | 404 |

### 排行榜
- 返回 total >= 1 + entries 数组

### Token 创建
- POST /api/token → 200 + token 8 位
