# 注册登录模块

## 覆盖范围

验证码发送（`POST /api/login/send-code`）+ 手机号登录（`POST /api/login`）+ token 绑定。

## 测试用例

### 发送验证码
| 用例 | 输入 | 预期 |
|------|------|------|
| 正常 | 11 位手机号 | 200 + expires_in=300 |
| 手机号太短 | 7 位 | 400 |
| 手机号含字母 | 1380013abcd | 400 |
| 空手机号 | "" | 400 |
| 无 phone 字段 | {} | 400 |
| 验证码写入 DB | 发送后查 DB | code 6 位 + used=0 |
| 重发替换旧码 | 发送两次 | DB 只保留 1 条 |

### 登录
| 用例 | 验证码 | 预期 |
|------|--------|------|
| MVP 万能码 | 888888 | 200 + success |
| 错误码 | 000000 | 400 |
| 真实 DB 码 | 从 DB 读取 | 200 + success |
| 已用码 | 二次登录 | 400 |
| 过期码 | 手动改 expires_at 到过去 | 400 |
| 手机号格式错 | "123" | 400 |

### Token 绑定
| 用例 | 预期 |
|------|------|
| 登录时传 token | user_tokens 表有记录 |
| 登录创建用户 | users 表有记录 |
| 不传 token 也正常 | 200 |
