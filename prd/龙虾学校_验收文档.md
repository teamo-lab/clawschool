## 一、用户动线总览

```
动线 A（朋友分享进入）：
  好友分享链接 → P1 分享落地页 → 点击 CTA → P3 安装命令弹窗 → P4 等待页 → P5 结果页

动线 B（直接访问进入）：
  访问首页 → P2 官网首页 → 点击 CTA → P3 安装命令弹窗 → P4 等待页 → P5 结果页

动线 C（分享成绩免费升级）：
  P5 结果页 → 点击"分享成绩免费升级 ~~¥19.9~~ ¥0" → 弹窗说明 → 点击"去分享"复制链接
  → 好友通过 /s/{token}?ref={sharer_token} 进入 → 好友完成测试
  → 分享者点击"好友已测试" → 后端验证 referral → 验证通过 → 直接进入升级流程（跳过支付）

动线 D（¥19.9 基础能力升级）：
  P5 → 点击"¥19.9 补齐基础能力" → P6 Step1(说明+付费) → Step1.5(扫码支付) → Step2(复制命令) → Step3(等待) → Step4(结果) → 回到 P5（数据刷新）

动线 E（¥99/月 高级能力订阅 — MVP 暂为 Waiting List）：
  P5 → 点击"¥99/月 高级能力订阅" → Waiting List 弹窗 → 输入手机号 → 提交 → 成功提示
  （完整付费流程代码已封存，下一期启用）

动线 F（个人主页）：
  P7 个人主页（需登录）→ 查看成绩+订阅状态 → 炫耀/重测/高级订阅
```
