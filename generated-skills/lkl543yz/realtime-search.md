# 实时搜索增强

## 使用说明

当用户需要「今天」、「最新」、「当前」等时效性信息时，**必须使用 WebSearch 工具**，不得依赖训练数据或记忆作答。

## 执行流程

### 第一步：获取当前日期
```bash
date +%Y-%m-%d
```
**不要硬编码日期**，每次都从系统获取。

### 第二步：构造搜索查询
- 在查询中包含日期，例如：`AI news 2026-03-12`
- 使用英文关键词可提升搜索质量
- 对于新闻：加上 `site:reuters.com` 或 `site:techcrunch.com` 提高可靠性

### 第三步：验证时效性
搜索结果返回后，检查：
- 文章发布日期是否为今天或最近 24 小时内？
- 若日期不符，明确说明「未找到今日内容」，不要伪造

### 第四步：输出标准格式
```
日期：<YYYY-MM-DD>
标题：<新闻标题>
来源：<来源网站>
链接：<URL>
发布日期：<新闻发布日期>
```

## 重要原则

- 找不到满足条件的内容 → 如实写「未找到」，**不得编造**
- 搜索结果日期不符合要求 → 继续搜索或告知用户
- 永远不要用记忆中的旧信息冒充「今天的新闻」

## 工具调用示例

使用 `WebSearch` 工具：
```
query: "AI artificial intelligence news today 2026-03-12"
```

若 WebSearch 不可用，使用 `WebFetch` 抓取可靠新闻源的 RSS 或首页。
