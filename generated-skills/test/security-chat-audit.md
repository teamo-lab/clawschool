# 群聊安全整理助手

## 使用说明

当需要对群聊记录进行安全整理时，按以下步骤执行：

## 执行流程

### 第一步：内容分类扫描

对输入的群聊内容，逐条识别以下风险类型：

| 风险类型 | 识别特征 | 处理建议 |
|----------|----------|----------|
| 敏感个人信息 | 手机号、身份证、银行卡号 | 标记并建议撤回或脱敏 |
| 可疑外链 | 短链、非官方域名、诱导点击 | 标记为高风险，建议不点击 |
| 隐私文件 | 截图含账号密码、内部文档 | 立即提醒发送者删除 |
| 钓鱼内容 | 中奖通知、紧急转账请求 | 标记为诈骗风险 |
| 违规内容 | 涉及法律红线的文字/图片 | 建议管理员处理 |

### 第二步：生成整理报告

输出格式：
```
## 群聊安全审查报告

**扫描时间**：[时间]
**消息总数**：[N] 条
**风险发现**：[M] 项

### 高风险项
- [消息摘要] → 风险类型：[类型] → 建议：[操作]

### 中风险项
- [消息摘要] → 风险类型：[类型] → 建议：[操作]

### 整理建议
1. [具体可执行的操作]
```

### 第三步：优先级排序

- 🔴 **立即处理**：含密码/账号/身份证等个人隐私
- 🟡 **今日处理**：可疑链接、内部文件外流
- 🟢 **记录备案**：一般性信息安全提示

## 注意事项

- 不存储或转发原始聊天内容
- 脱敏处理后再输出报告（如 `138****8888`）
- 发现高风险内容时，优先提醒用户而非自动操作
